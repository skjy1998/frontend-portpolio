# 지도 기반 장소 공유 서비스 (Frontend)

지도 기반으로 장소를 탐색하고 공유할 수 있는 **실제 운영 중인 서비스**의 프론트엔드 프로젝트입니다.  
사용자는 지도를 통해 장소를 한눈에 확인하고, 클러스터 및 리스트 UI를 통해 효율적으로 탐색할 수 있습니다.

> ⚠️ 본 프로젝트는 회사 상용 서비스로, **소스 코드는 비공개**이며  
> 본 저장소에는 프로젝트 설명 및 화면 예시만 포함되어 있습니다.

---

## 📌 프로젝트 개요

- **형태**: 지도 기반 장소 공유 웹 서비스
- **목적**: 사용자들이 특정 장소를 지도에서 쉽게 탐색하고 저장·공유할 수 있도록 제공
- **운영 여부**: 실제 서비스 운영 중
- **기여 범위**: 프론트엔드 단독 개발  
  (※ 디자인 시안은 제공받아 구현)

---

## 🧑‍💻 담당 역할 (Frontend)

- 지도 기반 UI 전체 구현
- 지도 마커 및 클러스터링 기능 구현
- 지도 인터랙션과 장소 리스트 UI 연동
- 장소 상세 화면 UI 구현
- 모바일 환경 중심 반응형 UI 개발
- API 연동 및 상태 관리

---

## 🛠 사용 기술

- **Framework**: Next.js
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Map**: Mapbox
- **State / Data**: API 기반 데이터 처리
- **Version Control**: Git

## ✍️ 프로젝트 설명

지도를 중심으로 한 UI 구조를 설계하여  
사용자가 **확대/축소, 클러스터 선택, 장소 탐색**을 자연스럽게 수행할 수 있도록 구현했습니다.

지도 상의 마커 밀집 구간에서는 클러스터링을 적용해 가독성을 높였고,  
클러스터 선택 시 해당 영역의 장소 리스트가 함께 표시되도록 구성하여  
지도와 리스트 간의 사용성을 강화했습니다.
